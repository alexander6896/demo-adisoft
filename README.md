# 🌮 Demo Landing Page — Sistemas Adisoft

Página web de demostración desarrollada por **Sistemas Adisoft** para mostrar a negocios locales cómo se vería su presencia digital. El ejemplo está basado en una taquería ficticia, pero el diseño y la estructura son adaptables a cualquier giro comercial.

---

## 🎯 Propósito

Esta demo es una herramienta de ventas. Se presenta a dueños de negocios que **no tienen página web** para mostrarles de manera tangible el valor de tener presencia en internet: aparecer en Google, recibir clientes por WhatsApp y transmitir profesionalismo.

El botón superior **"Quiero la mía →"** redirige directamente al WhatsApp de Sistemas Adisoft para iniciar la conversación de venta.

---

## ✨ Características de la demo

- **Barra de demo** fija en la parte superior con llamada a la acción directa a WhatsApp
- **Navbar** con scroll inteligente y menú de navegación
- **Hero** con badge de calificación Google Maps y reseña destacada
- **Barra de estadísticas** (años de experiencia, calificación, clientes diarios, platillos)
- **Sección de menú** con 6 platillos, imágenes y precios
- **Sección "Nosotros"** con historia del negocio y características clave
- **Testimonios** de clientes con calificación de 5 estrellas
- **Sección de contacto** con horarios, ubicación y formulario que envía mensaje por WhatsApp
- **Botón flotante de WhatsApp** con tooltip y animación
- **Footer** con crédito a Sistemas Adisoft
- Diseño **completamente responsivo** para móvil y escritorio
- Animaciones suaves al hacer scroll (IntersectionObserver)

## 🛠️ Tecnologías utilizadas

| Tecnología | Uso |
|---|---|
| HTML5 semántico | Estructura de la página |
| CSS3 puro | Estilos, animaciones, grid, responsive |
| JavaScript vanilla | Interactividad sin dependencias |
| Google Fonts | Tipografías Playfair Display + DM Sans |
| WhatsApp API (wa.me) | Formulario de contacto y botón flotante |
| Git & GitHub | Control de versiones y hosting con GitHub Pages |

Sin frameworks, sin dependencias npm. Carga rápida y fácil de mantener.

## 🔧 Cómo adaptar la demo a otro negocio

1. Cambia el nombre del negocio en `index.html` (búsqueda: `La Taquería del Centro`)
2. Actualiza los platillos/servicios en la sección `#menu`
3. Modifica colores en `styles.css` dentro del bloque `:root { ... }`
4. Cambia el número de WhatsApp del negocio en los enlaces `wa.me/52XXXXXXXXXX`
5. Actualiza horarios y dirección en la sección `#contacto`
